# Deep Research Agent

Evidence-first multi-agent research system.

## Pipeline

Coordinator -> Source Hunter -> Historical Research -> Verifier -> Perspective Comparison -> Memory -> Report

## Principles

- Claims require sources.
- Evidence and interpretation remain separated.
- Confidence is derived from observable factors.
- Reports preserve uncertainty and disagreement.

## V1.5 Build

Includes the foundation for:

- LangGraph agent workflow
- Scientific source retrieval
- Historical research module
- Perspective comparison
- Optional semantic memory
- SQLite research history
- CLI and Streamlit interfaces
